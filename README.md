# House_Manage

好客房租是直接促成房东与租户对接的生活服务平台，它包含房东发布房源，租户多维度寻找房源，智能匹配房源，近期行情查询等功能，减少中间环节产生的费用，提高房东与租户匹配的成功率.

采用了SOA架构思想进行设计，采用SpringBoot、SpringMVC、Mybaits、Dubbo等技术框架实现，好客租房直接促成房东与租户对接的生活服务平台。
